Frequently Asked Questions

Anthias is free, open-source digital signage management software built by Screenly, Inc. Screenly is a paid digital signage solution that is better suited for professional installations with multiple screens. You can learn more about the differences here. Screenly is a better choice if you need to manage multiple screens from a single account, cloud-based screen management, enterprise-grade user management, or professional support.

Anthias is the right choice if you are comfortable managing each screen individually, familiar with Linux, and do not need support, hosting, or security updates. Screenly is a better fit when you have more than one screen, want an out-of-the-box solution, or need support, cloud hosting, and security updates.

Anthias runs on Raspberry Pi single-board computers and 64-bit x86 PCs. Disk images are available for Raspberry Pi 2 through Raspberry Pi 5 (Pi 2 and Pi 3 are in maintenance mode — we recommend upgrading to Pi 4 or later). In addition to a Raspberry Pi or PC, you will need a screen that accepts HDMI input. Learn more about how to get started on our GitHub page.
