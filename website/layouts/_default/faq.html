FAQ

Common questions about Anthias.

Anthias is free, open-source digital signage software built by Screenly, Inc. Screenly is a paid, cloud-managed digital signage platform designed for professional multi-screen deployments. You can learn more about the differences on screenly.io.

Choose Screenly if you need multi-screen management from a single account, enterprise user management, or professional support. Choose Anthias if you're comfortable managing individual devices and want a free, self-hosted solution.

Anthias runs on Raspberry Pi 2 through 5 and 64-bit x86 PCs (Intel NUC or similar). You'll also need a screen with an HDMI input. We recommend a Raspberry Pi 4 or 5 for new installations.

Anthias is a community-supported project. You can get help on the Anthias forum or file issues on GitHub. For professional support, SLAs, and managed infrastructure, consider Screenly.

No. Anthias runs entirely on your local network. You only need internet access for the initial installation and to display content from external URLs. Uploaded images and videos play from local storage.

Each Anthias instance manages one screen. You can run multiple instances on separate devices, but each has its own dashboard. If you need centralized multi-screen management, Screenly is designed for that.

If you installed via balenaHub, updates are applied automatically over the air. For command-line installations, re-run the installer or pull the latest Docker images. Disk image installations are pinned to the version you downloaded.
